# DocsFlip

DocsFlip is a platform for East African businesses and communication agencies to convert PDF documents into interactive, AI-assisted content.

## Repository Purpose
This repository contains:
- Product vision and MVP scope
- Technical implementation guidance
- AI agent kickoff prompts for no-code / AI coding agents

## Key Files
- `Docsflip_AI_Agent_Kickoff_Prompt.md` – Primary instructions for AI coding agents
- `Docsflip_MVP_Product_Brief_v2.md` – Product scope, features, and constraints
- `Docsflip_Technical_Implementation_Guide.md` – Architecture and technical decisions

## How to Use (For AI Coding Agents)
1. Read all markdown files in the root directory
2. Treat them as the single source of truth
3. Ask for clarification before making assumptions
4. Do not introduce features not defined in the product brief
